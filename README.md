# מפגש הספורט טבריה — מערכת יצירת סטורי זכיות

אפליקציית ווב מובייל-first (RTL עברית) שיוצרת **סטורי זכיות בגודל 1080×1920** לפרסום בפייסבוק/אינסטגרם — הכול בצד לקוח, מיידי, ללא שרת.

## עיקרון מפתח: התמונה המקורית לא משתנה

התמונה שהמשתמש מעלה נשארת **שכבת תמונה שלמה** (`<img>` עם `object-fit: contain`) — לא נחתכת, לא נערכת, לא עוברת AI, הפיקסלים לא משתנים. מותר רק להקטין/להגדיל אותה כיחידה אחת, למקם אותה, ולהוסיף מסביבה מסגרת/רקע/טקסטים.

## עברית תקינה (ללא היפוך)

כל הטקסטים נבנים כ-**DOM אמיתי** עם `direction: rtl` + `unicode-bidi: plaintext`. הייצוא ל-PNG נעשה דרך **SVG `foreignObject`** — הדפדפן עצמו מרנדר את ה-HTML, כך שהעברית לא מתהפכת אף פעם. **לא** משתמשים בציור טקסט על Canvas.

## מבנה הקבצים

| קובץ | תפקיד |
|------|-------|
| `index.html` | כל האפליקציה — ממשק + מנוע DOM + ייצוא foreignObject→PNG |
| `manifest.webmanifest` | PWA (הצמדה למסך הבית) |
| `sw.js` | Service Worker (אופליין; network-first ל-HTML) |
| `icon.svg` | אייקון האפליקציה |
| `PROJECT_BRIEF.md` | בריף המקור |

## זרימת שימוש

1. מעלים / מצלמים תמונת זכייה
2. מזינים סכום (רשות) ובוחרים סוג זכייה
3. בוחרים **סגנון עיצוב** (6 סגנונות)
4. בוחרים **כותרת** ו**משפט סיום** (מוכנים או מותאם אישית)
5. "צור סטורי" → תצוגה מקדימה חיה
6. הורדה כ-PNG (`winning_story_YYYYMMDD_HHMM.png`) או שיתוף

| שם | שם פנימי | אופי |
|----|----------|------|
| שחור זהב יוקרתי | `luxury_gold` | רקע שחור, מסגרת זהב זוהרת, קונפטי |
| צ'אנס חגיגי | `chance_gold` | זהב/שחור, סמלי קלפים, מזל |
| חיש גד צבעוני | `hishgad_festive` | צבעוני, נצנצים, זהב+כסף |
| ווינר ספורטיבי | `winner_sport` | כחול/שחור/זהב, תאורת אצטדיון, כדורגל |
| מסגרת אמינות | `authenticity_frame` | מינימלי, תמונה גדולה, "תמונה אמיתית" |
| יום של זכיות | `multiple_wins` | חגיגי, כותרת גדולה, "סכום כולל" |

## מסגרות מותאמות אישית (AI)

אפשר לייצר מסגרות/רקעים בכל מחולל AI (Midjourney, DALL·E וכו') בגודל 1080×1920 ולהעלות אותם דרך "➕ הוסף מסגרת". זה **לא** סותר את כלל "ללא AI" — הכלל מגן על תמונת הזכייה האמיתית; המסגרת היא אמנות דקורטיבית נפרדת.

- **שתי דרכי שילוב:** *רקע מאחורי התמונה* (JPG רגיל) או *מסגרת שקופה מעל* (PNG עם חלון שקוף).
- **מיקום חופשי:** גוררים את התמונה למיקום, ומשנים גודל במחוון / גלגלת / צביטה בשתי אצבעות (object-fit: contain — התמונה לא נחתכת).
- **הצג/הסתר טקסטים:** אם המסגרת כבר כוללת אזורי טקסט, אפשר לכבות כותרת/סכום/סוג/משפט/מיתוג.
- **ספריית מסגרות:** נשמרת ב-localStorage במכשיר (רקעים נדחסים ל-JPEG לחיסכון מקום; שקופות נשמרות כ-PNG). מחיקה דרך ✕ על האריח.
- **טיפ לפרומפט:** בקש `ornate luxury gold frame, dark background, empty clear center for a photo, decorative corners, vertical 9:16, leave top and bottom clear for text`.

## מבנה הסטורי

כותרת גדולה → סכום זכייה (אם הוזן) → תגית סוג זכייה → התמונה המקורית במסגרת → משפט סיום → מיתוג קבוע: **מפגש הספורט טבריה · שד' ספיר, טבריה**.

## הרצה מקומית

אתר סטטי, ללא build:

```bash
npx serve .
```

> הערה: PWA/אופליין (`sw.js`) פועל רק תחת `http(s)://`. הייצוא ל-PNG (foreignObject) עובד מצוין ב-Chrome (דסקטופ ואנדרואיד) וב-Safari מודרני.

## פריסה (חינם)

מעלים את התיקייה כמו שהיא ל-Netlify / GitHub Pages / Cloudflare Pages. אין שלב build.
