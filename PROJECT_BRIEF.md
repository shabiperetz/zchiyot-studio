# מפגש הספורט טבריה — מערכת עיצוב תמונות זכיות

בריף פתיחה. כשמתחילים לעבוד על הפרויקט — קרא את הקובץ הזה קודם, ואז המשך בבנייה.

## מה בונים

מערכת שמעצבת תמונות זכיות להעלאה יומית לרשתות החברתיות (טוטו + מפעל הפיס).
הזרימה שהמשתמש רוצה: מעלה תמונה → רושם סכום זכייה → בוחר סוג משחק → **התמונה מתעצבת מיד** → הורדה/שיתוף.
דגש עליון: **מהירות** — כל התהליך תוך שניות מהטלפון.

## שם העסק

**מפגש הספורט טבריה** (יופיע על כל תמונה; ניתן לעריכה/שינוי במערכת).

## החלטות שכבר סוכמו (מול המשתמש)

1. **פלטפורמה:** אפליקציית ווב מבוססת טלפון (מובייל-first), אפשר להצמיד למסך הבית (PWA). RTL עברית.
2. **תבניות:** כמה תבניות קבועות (3-5) שנעצב מראש; המשתמש בוחר בכל פעם.
3. **מיתוג:** יש רק שם עסק. אין לוגו/צבעים קיימים → לעצב זהות בסיסית. כיוון מוסכם: **ירוק־זהב** (מזל/זכייה).
4. **עיצוב התמונה:** עם תבניות Canvas (מיידי, חינמי, אמין, התמונה נשארת מקומית) — **לא** ייצור תמונה ב-AI.
5. **משפטי ברכה:** מתחלפים לפי גובה הזכייה. ב-MVP — משפטים מוכנים שמתחלפים; אפשר להוסיף ייצור AI לטקסט בהמשך (Claude API), לא לתמונה.

## סטאק מומלץ ל-MVP

התחלה הכי מהירה: **קובץ HTML יחיד עם Canvas** (בלי build, נפתח בכל טלפון, עובד אופליין, התמונה לא עוזבת את המכשיר).
אם נרצה להרחיב אחר כך: Vite + React + PWA, אירוח חינם ב-Netlify/GitHub Pages.

## תכולת ה-MVP

- העלאת תמונה (כולל צילום ישיר מהמצלמה במובייל)
- שדה סכום זכייה + בחירת סוג משחק (לוטו / דאבל לוטו / צ'אנס / 777 / חישגד / טוטו / ווינר / קסם)
- שדה משפט חופשי (רשות)
- בורר תבנית (3 תבניות)
- תצוגה מקדימה חיה על Canvas בריבוע 1080×1080 (אינסטגרם/וואטסאפ)
- שם העסק "מפגש הספורט טבריה" (נשמר ב-localStorage, ניתן לעריכה)
- כפתור הורדה + שיתוף (Web Share API)

## הערות

- כל העיבוד בצד לקוח — פרטיות: תמונת הכרטיס/הזוכה לא נשלחת לשום שרת.
- זהות ויזואלית לעצב: ירוק עמוק + זהב, טיפוגרפיה נקייה, תחושת יוקרה/חגיגה.
