# תיקיית מסגרות מובנות

האפליקציה טוענת אוטומטית את כל המסגרות שברשומות בקובץ `frames.json`.
**השמות של הקבצים כבר לא משנים** — אפשר להשאיר שמות של ChatGPT (עם רווחים/פסיקים/סוגריים).

## איך מוסיפים מסגרות חדשות

1. שים את קבצי התמונה כאן בתיקייה (`assets/frames`).
2. לחיצה כפולה על **`build-frames-manifest.ps1`** (או לחיצה ימנית → Run with PowerShell).
   - אם חלון כחול מבקש אישור: "More info" → "Run anyway".
3. זהו — פתח מחדש את האפליקציה והמסגרות יופיעו.

הסקריפט חכם:
- כולל רק תמונות אנכיות (יחס קרוב ל-9:16) — מדלג על ריבועים/קולאז'.
- מדלג על כפילויות (אותה תמונה שהורדה פעמיים).
- מסמן `overlay` אם השם כולל `overlay`/`transparent`, אחרת `bg` (רקע).
- **יוצר תמונות מוקטנות** (`thumbs/`, JPEG ~18KB) כדי שבורר הסגנונות ייטען מהר. הקובץ המלא נטען רק כשבוחרים את המסגרת (לתצוגה ולייצוא).

## שמות תצוגה בעברית

6 הקבצים הראשיים מקבלים שם עברי יפה (מוגדר ב-`index.html`, `FRAME_NAME_MAP`):
`01_black_gold_luxury` → שחור-זהב יוקרתי · `02_green_gold_brand` → ירוק-זהב ·
`03_chance_cards_gold` → צ'אנס · `04_colorful_hishgad` → חיש גד ·
`05_winner_sport_blue_gold` → ווינר · `06_minimal_trust_gold_frame` → אמינות.
כל השאר מקבלים "מסגרת 1", "מסגרת 2"...

## הערות
- מסגרות מובנות אינן ניתנות למחיקה מהממשק. כדי להסיר — מחק את הקובץ מהתיקייה והרץ שוב את הסקריפט.
- מסגרות שמעלים דרך "➕ הוסף מסגרת" נשמרות ב-localStorage המקומי (וניתנות למחיקה).
- פרומפטים ליצירה: `../../FRAME_PROMPTS.md`.
